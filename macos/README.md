## 자동 설치

Brewfile

> [!NOTE]
> Brewfile mas<br/>
> mas는 맥앱스토어를 커맨드라인으로 사용할 수 있게하는 툴입니다. 인스톨과 업데이트만 가능합니다. homebrew는 mas를 설치하는 방법중 하나일 뿐 입니다. homebrew에서 설치한 mas는 uninstall할 수 있지만, mas로 설치한 앱은 homebrew와 무관합니다.
